# Contact Book Web App

A simple web-based contact book application for managing personal or professional contacts. This app allows users to add, view, edit, and delete contact information through a user-friendly interface.

## Features

- Add new contacts with details
- View all saved contacts
- Edit existing contact information
- Delete contacts
- User authentication (login/logout)
- Responsive and clean UI

## Technologies Used

- PHP (Backend)
- HTML, CSS, JavaScript (Frontend)
- MySQL (Database)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/chamodrandeni/contact-book-web-app.git
   ```
2. Import the `contactbook.sql` file from the `db/` directory into your MySQL database.
3. Update the database credentials in `db_connection.php`.
4. Run the app on your local server (e.g., XAMPP, WAMP, or LAMP).

## Folder Structure

- `bkp/` - Backup HTML templates
- `db/` - Database schema
- `icons/` - App icons
- Main PHP and HTML files in the root directory
